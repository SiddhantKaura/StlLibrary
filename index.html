Vector

Present in #include<vector> header file.

It is like dynamic array which can resize itself during insertion and deletion of element.

Function Name Function Calling Description
vector::size vector_name.size() gives you the size of the vector.
vector::push_back vector_name.push_back(element) is used to add element at the back of the vector.
vector::insert vector_name.insert(position,element) is used to insert elements at the given position in the vector.
vector::begin vector_name.begin() is used to specify the beginning position of the vector.
vector::end vector_name.end() is used to specify the end position of the vector.
vector::erase vector_name.erase(position of element to be erased) is used to erase the element or group of elements by specifying their position in the vector.

Algorithm

Present in #include<algorithm> header file.

It is a header file which contains many useful functions.

Function Name Function Calling Description
find container_name.find(startpos,endpos,elementTobeFound) returns the first position of element to be found/if not found then returns end of conatiner.
count container_name.count(startpos,endpos,element) returns total count of element in the conatiner.
binary_search container_name.binary_search(startpos,endpos,element) returns true if element found otherwise false.
sort container_name.sort(startpos,endpos) is used to sort the container from starting position to ending position.
min min(a,b) is used to find the minimum of two elements such as a,b.
max max(a,b) is used to find the maximum of two elements such as a,b.
lower_bound lower_bound(startpos,endpos,element) is used to find the position of first element in container which is equal to the element or greater.
upper_bound upper_bound(startpos,endpos,element) is used to find the position of first element in container which is strictly greater then the element.
swap swap(a,b) is used to swap position of two elements such as a,b.
